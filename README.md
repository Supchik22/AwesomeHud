# AwesomeHud :mushroom:
AwesomeHUD is a simple client-side Minecraft mod that adds a useful hud to the screen.

## Features:
+ Compass
+ Recovery Compass
+ Clock
+ Armor Display


### How is it work:
>There is no config file, you need to edit the source code instead
